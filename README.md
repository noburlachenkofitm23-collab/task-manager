# Task Manager

Мінімалістичний вебзастосунок для управління завданнями з динамічною пріоритезацією.

## 🚀 Технології
- **Frontend:** React, JavaScript, Tailwind CSS
- **Backend/DB:** Firebase Firestore
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🛠 Функціонал
- Створення, редагування, видалення завдань.
- Динамічна пріоритезація (High/Medium/Low) на основі часу до дедлайну.
- Автоматичний перерахунок пріоритетів без перезавантаження.
- Хмарне збереження даних (Firebase).
- Темна та світла теми.

## 💻 Як запустити проєкт
1. Клонуйте репозиторій:
   `git clone [посилання на твій репозиторій]`
2. Встановіть залежності:
   `npm install`
3. Створіть файл `.env` на основі `.env.example` та додайте свої Firebase API ключі.
4. Запустіть проєкт:
   `npm run dev`
